
- 哈希表

```javascript
const twoSum = (nums, target) => {
    const map = new Map();
    
    for (let index = 0; index < nums.length; index++) {
      const other = target - nums[index];
      if (map.has(other)) {
        return [map.get(other), index];
      } else {
        map.set(nums[index], index);
      }
    }
}
```
